Condicionais no JavaScript

As condicionais servem para tomar decisões dentro do código, fazendo com que ele apenas execute se algo for de acordo com o que a condicional pede no bloco.
As condicionais são as padrões:
IF: Caso algo seja favorável, executa
ELSE: Caso algo não seja favorável, executa outra parte do bloco.
ELSE IF: Caso algo não seja favorável, pede outra ideia, se for favorável, executa.
SWITCH: Executa o caso que for favorável dentre várias funções.

Exemplo de condicional "if"

<script>
let idade = 18;

if (idade >= 18) {
  console.log("Maior de idade");
}
</script>

Exemplo de condicional "else"

<script>
let idade = 16;

if (idade >= 18) {
  console.log("Maior de idade");
} else {
  console.log("Menor de idade");
}
</script>

Exemplo de condicional "else if"

<script>
let nota = 7;

if (nota >= 9) {
  console.log("Excelente");
} else if (nota >= 6) {
  console.log("Aprovado");
} else {
  console.log("Reprovado");
}
</script>

Exemplo de condicional "switch"

<script>
let dia = 3;

switch (dia) {
  case 1:
    console.log("Domingo");
    break;

  case 2:
    console.log("Segunda-feira");
    break;

  case 3:
    console.log("Terça-feira");
    break;

  default:
    console.log("Dia inválido");
}
</script>
