Document Object Model

\

O DOM pega as informações da página HTML e transforma em uma "árvore", que permite que o script manipule a estrutura e conteúdo da página.
Com o DOM é possível acessar elementos, alterar conteúdo, mudar estilos e reagir a cliques.

getElementById()

Serve para achar um elemento pelo ID.

const el = document.getElementById('btn');

querySelector()

Seleciona o primeiro elemento usando seletor CSS.

const el = document.querySelector('.box');

querySelectorAll()

Seleciona todos os elementos do tipo indicado.

const itens = document.querySelectorAll('.item');

value

Pega o valor digitado em um input.

let valor = input.value;

checked

Verifica se um input está marcado.

let ativo = checkbox.checked;

checkValidity()

Verifica se um input é válido.

input.checkValidity();

textContent

Lê ou altera o texto de um elemento.

elemento.textContent = "Novo texto";

style

Permite alterar o CSS via JavaScript.

elemento.style.color = "red";

classList

Manipula classes de um elemento.

elemento.classList.add("ativo");
elemento.classList.remove("ativo");

addEventListener()

Adiciona um evento a um elemento.

botao.addEventListener("click", function() {
  console.log("Clicou");
});

DOMContentLoaded

Executa quando o HTML termina de carregar.

document.addEventListener("DOMContentLoaded", function() {
  console.log("Carregou");
});

createElement()

Cria um elemento HTML via JavaScript.

const div = document.createElement("div");

appendChild()

Adiciona um elemento na página.

document.body.appendChild(div);

remove()

Remove um elemento da página.

elemento.remove();
