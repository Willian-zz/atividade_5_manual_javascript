O que é uma variável

Uma variável é uma espécie de "marca" na memória do computador, que é usada para armazenar e manipular dados, como números, textos e/ou até objetos

Como declarar uma variável em JavaScript

Para declarar uma variável em JavaScript é necessário utilizar "const", "let" ou "var", isso vai definir de qual forma a variável vaise comportar. Depois é preciso adicionar o nome dela, pode ser qualquer nome que seja possível escrever e relembrar de forma fácil. Agora se você quiser acessar algo do seu arquivo HTML é preciso utilizar "document.", se não, não é necessário. E depois disso, se for preciso utilizar "document." é necessário achar o "id" ou "class" que foi classificada ao elemento, usando de "getElementById" para objetos classificados com "id", ou ".getElementByClass" para elementos classificados com "class", ou em casos que é necessário acessar qualquer tipo de elemento, se usa "querySelector"

Exemplo de variável que acessa elementos HTML:

const botao = document.getElementById('btn'); OU const botao = document.getElementByClass('btn')

Exemplo de variável com querySelector:

const texto = document.querySelector('#text'); OU const texto = document.quertSelector('.text'); OU const texto = document.querySelector('p')

Exemplo de variável simples:

let numero = 0; OU var numero = 0;

Diferença entre var, let e const

O "const" é para o variáveis que não mudam durante a execução do programa, por exemplo um "button" ou uma "div".
Já o "let" é para variáveis de tipo que mudam durante a execução, por exemplo o resultado de uma soma, ou o nome de do usuário.
O "var" éuma variável mais antiga, que tem a possibilidade de ser acessada de qualquer lugar dentro da função, então se por exemplo, ela for declarada dentro de um "if", depois ela pode ser acessada fora dele também.

Escopo global

Um escopo é a variável do seu código, e como o nome já diz, o escopo global, pode ser acessado em qualquer função durante o execução do seu programa, nãose limitando a apenas uma função.

let numero = 0;

Escopo de função

Um escopo de função, ao invés do global, ele não pode ser acessado durante todo o código, apenas dentro de uma função especifica, então a variável deve ser declarada dentro da função para ser considerado um escopo desse tipo.

function soma(){
let numero = 0;
numero++;
console.log(numero);
}

Escopo de bloco

Um escopo de bloco é o mais restrito de todos, funcionado apenas dentro de um bloco(if, while, for e etc...), existindo apenas dentro dela.
