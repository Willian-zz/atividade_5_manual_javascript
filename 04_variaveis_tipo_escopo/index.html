Variável

Uma variável é uma espécie de "marca" na memória do computador, que é usada para armazenar e manipular dados, como números, textos e/ou até objetos

Como declarar uma variável em JavaScript

Para declarar uma variável em JavaScript é necessário utilizar "const", "let" ou "var", isso vai definir de qual forma a variável vaise comportar. Depois é preciso adicionar o nome dela, pode ser qualquer nome que seja possível escrever e relembrar de forma fácil. Agora se você quiser acessar algo do seu arquivo HTML é preciso utilizar "document.", se não, não é necessário. E depois disso, se for preciso utilizar "document." é necessário achar o "id" ou "class" que foi classificada ao elemento, usando de "getElementById" para objetos classificados com "id", ou ".getElementByClass" para elementos classificados com "class", ou em casos que é necessário acessar qualquer tipo de elemento, se usa "querySelector"

Exemplo de variável que acessa elementos HTML:

<script>
const botao = document.getElementById('btn');
const botoes = document.getElementsByClassName('btn');
</script>

Exemplo de variável com querySelector:

<script>
const texto = document.querySelector('#text');
const classe = document.querySelector('.text');
const paragrafo = document.querySelector('p');
</script>

Exemplo de variável simples:

<script>
let numero = 0;
OU
var numero = 0;
</script>

Diferença entre var, let e const

const

Valor fixo

Usado para variáveis que não mudam durante a execução.
Exemplo: elementos HTML como "button" ou "div".

let

Valor variável

Usado para valores que podem mudar durante o programa.
Exemplo: resultados de contas ou dados do usuário.

var

Escopo antigo

É uma forma antiga de declarar variáveis.
Pode ser declarada dentro de um bloco "if", e ser acessada dentro e fora dele.

Tipos de escopo

Escopo global

Pode ser acessado em qualquer parte do código, não se limitando a apenas uma função ou bloco.

let numero = 0;

console.log(numero);

Escopo de função

Só pode ser acessado dentro da função onde foi declarado.

function soma() {
  let numero = 0;
  numero++;
  console.log(numero);
}

Escopo de bloco

Funciona apenas dentro de blocos como if, for ou while.

if (0 > 1) {
  let numero = 0;
  numero++;
  console.log(numero);
}
