Repetição no JavaScript

As repetições servem para automatizar tarefas, elas fazem o bloco executar até que uma condição seja favorável. As estruturas de repetição são:
FOR: É um laço ideal para uma contagem especifica.
WHILE: É uma laço para repetição indeterminada. Enquanto for favorável, executa.
DO ... WHILE: Assim como o "While" é indeterminado, mas executa ao menos uma vez.

Exemplo de repetição "for"

<script>
for (let i = 1; i <= 5; i++) {
  console.log("Número: " + i);
}
</script>

Exemplo de repetição "while"

<script>
let i = 1;

while (i <= 5) {
  console.log("Contador: " + i);
  i++;
}
</script>

Exemplo de repetição "do ... while"

<script>
let i = 1;

do {
  console.log("Valor: " + i);
  i++;
} while (i <= 5);
</script>
