Operadores e comparações

No JavaScript existem vários tipos de operadores, como os clássicos operadores aritméticos ou matemáticos, de comparação, os lógicos e os de atribuição.

Aritméticos

São usados para cálculos numéricos.

"+" Soma
"-" Subtração
"*" Multiplicação
"/" Divisão
"%" Resto de divisão

Comparação

Retornam true ou false.

"==" Compara valor ignorando o tipo
"===" Compara valor de mesmo tipo
"!=" Diferente ignarando o tipo
"!==" Diferente de mesmo tipo
">" Maior que
"<" Menor que
"<=" Maior igual que
"<=" Menor igual que

Lógicos

São usados para combinar condições, mas tambem retornam true ou false.

"&&" "AND" ou "E", vai retornar true se todas as opções forem favoráveis.
"||" "OR" ou "OU", retorna true se ao menos uma for favorável.
"!" "NOT" ou "NÃO", inverte o valor (true para false).

Atribuição

Servem para definir ou mudar valores de variáveis.

"=" Atribuição simples.
"+=" Soma e atribui
"-=" Subtrai e atribui
"*=" Multiplica e atribui
"/=" Divide e atribui

Exemplo de atribuição "aritmética"

<script>
let a = 10;
let b = 5;

console.log(a + b); //Retorna 15
console.log(a - b); //Retorna 5
console.log(a * b); //Retorna 50
console.log(a / b); //Retorna 2
console.log(a % b); //Retorna 0
</script>

Exemplo de atribuição "comparativa"

<script>
let x = 10;
let y = "10";

console.log(x == y); //Retorna true
console.log(x === y); //Retorna false
console.log(x != y); //Retorna false
console.log(x !== y); //Retorna true
</script>

Exemplo de atribuição "lógica"

<script>
let idade = 20;

console.log(idade > 18); //Retorna true
console.log(idade < 18); //Retorna false
console.log(idade >= 18); //Retorna true
console.log(idade <= 18) //Retorna false;
</script>

Exemplo de atribuição

<script>
let temCarteira = true;
let idade = 20;

console.log(idade >= 18 && temCarteira); //Retorna true
console.log(idade >= 18 || temCarteira); //Retorna true
console.log(!temCarteira); //Retorna false
</script>
