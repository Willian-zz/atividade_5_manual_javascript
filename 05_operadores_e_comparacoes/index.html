Operadores e comparações

No JavaScript existem vários tipos de operadores, como os clássicos operadores aritméticos ou matemáticos, de comparação, os lógicos e os de atribuição.

Aritméticos

São usados para cálculos numéricos.

"+" Soma
"-" Subtração

Comparação

Retornam true ou false.

AAA

Lógicos

São usados para combinar condições, mas tambem retornam true ou false.

AAA

Atribuição

Servem para definir ou mudar valores de variáveis.

AAA

<script>
let a = 10;
let b = 5;

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(a / b);
console.log(a % b);
</script>

<script>
let x = 10;
let y = "10";

console.log(x == y);
console.log(x === y);
console.log(x != y);
console.log(x !== y);
</script>

<script>
let idade = 20;

console.log(idade > 18);
console.log(idade < 18);
console.log(idade >= 18);
console.log(idade <= 18);
</script>

<script>
let temCarteira = true;
let idade = 20;

console.log(idade >= 18 && temCarteira);
console.log(idade >= 18 || temCarteira);
console.log(!temCarteira);
</script>
