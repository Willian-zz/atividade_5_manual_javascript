O que é JavaScript

O JavaScript é uma linguagem de programação de alto nível, interpretada e dinânmica, ela é essencial para o desenvolvimento web.
O JavaScript tem vários frameworks e bibliotecas que ajudam a modificar o site sem ter que escrever uma função nova totalmente do zero.

Para que serve JavaScript

É uma linguagem muiot importante para o desenvolvimento web, serve tanto para o back-end quanto para o front-end, deixando o site mais dinâmico e funcional.
O JavaScript serve tanto para deixar o site mais bonito, mas serve para deixar ele funcional como, por exemplo, validar formulários.

Como o JavaScript complementa HTML e CSS

O JavaScript ajuda o HTML e CSS deixando de ser uma página estática e transformando em uma aplicação mais interativa e mais funcional.
Apenas com o HTML e CSS não é possível fazer formulários funcionais, também não é possível fazer muitas funções como calcular valores.
