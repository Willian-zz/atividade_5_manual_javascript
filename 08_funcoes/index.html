O que é uma função

Uma função é uma um elemento que contém, como o nome diz, uma função, sendo utilizado para organizar, otimizar tempo e facilitar a leitura do código, além de minimizar a estrutura.

Declaração de função

Para declarar uma função é preciso adicionar "function" no inicio do comando, depois disso é preciso adicionar o nome da função, em seguida abrir o parametro por meio de "()".

Exemplo de função sem parâmetro:

function soma(){
let somar = 0;
somar = 5 + 3
console.log(somar);
}

Exemplo de função parâmetro:

function soma(x,y){
let somar = 0;
somar = x + y
console.log(somar);
}

Exemplo de função de retorno:

function soma(x,y){
return x + y;
}
